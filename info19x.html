Arduino RFID Door Lock System

Abstract

This project demonstrates a simple RFID access control system using an Arduino Uno and an MFRC522 RFID module. When an authorized RFID card is detected, the system activates a servo motor to simulate unlocking a door and turns on a green LED. If an unauthorized card is scanned, a red LED indicates access denial.

Objective

  • Build a basic RFID security system
  • Learn RFID communication with Arduino
  • Control physical devices based on authentication

Components

  • Arduino Uno
  • MFRC522 RFID Module
  • Servo Motor
  • Green LED
  • Red LED
  • Resistors
  • Breadboard and jumper wires

Working Principle

The MFRC522 RFID module reads the UID of an RFID card. The Arduino compares the UID with a predefined authorized UID stored in the code. If the UID matches, the servo motor rotates to unlock the mechanism and the green LED turns on. If the UID does not match, the system denies access and activates the red LED.

Code


#include <SPI.h>       // Library for SPI communication with the RFID module
#include <MFRC522.h>      // Library for working with RC522 RFID modules
#include <Servo.h>       // Library for controlling servo motors

// Define pins for the RFID module
#define SS_PIN 10        // Slave Select (CS) pin for RFID
#define RST_PIN 9        // Reset pin for RFID

MFRC522 mfrc522(SS_PIN, RST_PIN); // Create an RFID object

Servo myServo;                     // Create a servo object

// LED and servo pins
int greenLED = 6;                  // Green LED for authorized access
int redLED = 5;                    // Red LED for denied access
int servoPin = 3;                  // Pin for connecting the servo

// Authorized card UID (example)
byte authorizedUID[4] = {0xDE, 0xAD, 0xBE, 0xEF};

void setup() {
    Serial.begin(9600);            // Start serial communication at 9600 baud
    SPI.begin();                   // Initialize SPI bus
    mfrc522.PCD_Init();            // Initialize the RFID reader

    pinMode(greenLED, OUTPUT);     // Set LED pins as outputs
    pinMode(redLED, OUTPUT);

    myServo.attach(servoPin);      // Attach servo to the defined pin
    myServo.write(0);              // Set servo to initial position (0°)

    Serial.println("Scan RFID card..."); // Prompt the user
}

void loop() {
    // If no new card is present, return and check again
    if (!mfrc522.PICC_IsNewCardPresent())
        return;

    // If the card cannot be read, return
    if (!mfrc522.PICC_ReadCardSerial())
        return;

    // Assume access is allowed initially
    boolean access = true;

    // Compare scanned card UID with authorized UID
    for (byte i = 0; i < 4; i++) {
        if (mfrc522.uid.uidByte[i] != authorizedUID[i]) {
            access = false; // Deny access if any byte doesn't match
        }
    }

    if (access) {
        digitalWrite(greenLED, HIGH); // Turn on green LED
        digitalWrite(redLED, LOW);    // Turn off red LED

        myServo.write(180);           // Rotate servo to open position
        delay(3000);                  // Keep it open for 3 seconds
        myServo.write(0);             // Return servo to closed position

        digitalWrite(greenLED, LOW);  // Turn off green LED
    } else {
        digitalWrite(redLED, HIGH);   // Turn on red LED for denied access
        delay(2000);                  // Keep it on for 2 seconds
        digitalWrite(redLED, LOW);    // Turn off red LED
    }

    // Halt communication with the card
    mfrc522.PICC_HaltA();
}
